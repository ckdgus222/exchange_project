# :computer: 프로젝트 소개

나라에 맞게 실시간 으로 환율을 보여주고 
원화로 값을 계산 합니다.

# :alarm_clock: 개발 기간
* 24.04.02 - 24.04.07

# :wrench: Tech

* React
* Vite
* Axios
* 공공 데이터 API


# :sunny: 주요 기능

* 실시간 환율.
* 각 나라에 환율을 적용해 원화 값 계산.
* 선택한 나라에 따라 여행지 추천.
* 좋아요와 댓글 기능.
